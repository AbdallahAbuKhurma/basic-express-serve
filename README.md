## Setup

### Installed dependencies

    npm i dotenv
    npm i express
    npm i -D jest supertest

### .env requirments

    * PORT - port number

### Running the application

    * npm start
    * Endpoint: **/person?name=name**

### Tests

    * Unit Tests: npm run test
    * Lint Tests: npm run lint
